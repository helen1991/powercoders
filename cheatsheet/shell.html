Shell cheatsheet.

Creating (making) directory.

  • Get back to your directory.
  • Creat a new directory called test.
  • Check it exist with dir or ls.
  • Use the GUl shell to see the directory.

Deleting directory.

  • Remove the directory you created in the previous step.

Action Windows OSX
